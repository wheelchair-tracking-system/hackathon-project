Credit & Code

This project was developed as part of a hackathon conducted at JECRC University. The goal of this project is to address the challenges faced in wheelchair management within hospitals, by providing an efficient system for real-time tracking, availability checking, and optimized usage of wheelchairs. Although this is just a prototype, it showcases the potential to improve hospital operations and enhance patient experience by ensuring wheelchairs are readily available when needed.
